# Oxi — Document Processing Suite

## This repository has moved to GitLab

**https://gitlab.com/Ryujiyasu/oxi**

Oxi is an open-source document rendering engine built with Rust + WebAssembly.  
View, render, and edit .docx / .xlsx / .pptx / PDF natively in the browser — no server required.

All development, issues, and contributions are now on GitLab.
